# Changelog

All notable changes to Rook will be documented in this file.

## [1.0.0] - 2026-04-14

### 🎉 Initial Release

#### Added
- 🔐 User authentication (JWT + bcrypt)
- 🖱️ One-click agent deployment (3-step wizard)
- 📊 Dashboard with real-time stats
- 💬 Built-in chat panel
- ⏰ Auto keep-alive via Cron-job.org
- 💓 Health monitoring with auto-restart
- 🤖 OpenRouter integration (free models)
- 🚀 Render.com deployment automation
- 🗄️ MongoDB Atlas auto-setup
- 📦 Agent templates (Personal Assistant, Customer Support, Code Helper)
- 📱 Messaging connection support (Telegram, WhatsApp, Discord)
- 📋 Activity feed
- 🔒 AES encrypted API key storage
- 🎨 Beautiful dark-mode dashboard (Next.js + Tailwind)
- 📡 Full REST API
- 🛡️ Rate limiting, Helmet, CORS
